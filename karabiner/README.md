# Karabiner Config

My personal Karabiner-Elements configuration using [karabiner.ts](https://github.com/evan-liu/karabiner.ts).

## Setup

```bash
npm install
npm run build
```

This generates the Karabiner configuration file at `~/.config/karabiner/karabiner.json`.

## Key Mappings

For the complete keyboard bindings reference, see [KEYBINDINGS.md](../KEYBINDINGS.md).

### Quick Overview

- **Caps Lock** → Hyper Key (⌘⌃⌥⇧) when held, Escape when tapped
- **Hyper + O** → Open applications
- **Hyper + W** → Window management
- **Hyper + V** → Vim navigation
- **Hyper + T** → Terminal shortcuts
- **Hyper + S** → System controls
- **Hyper + R** → Raycast commands
- **Hyper + B** → Browse websites
